### College Life App

A fully-functional mobile application built with **Kivy** and **Python** using **MVC architecture** to help college students manage their daily academic life.
All user data is saved locally using **JSON files**.

Demo: https://drive.google.com/file/d/1XwN_Chx5A4eARYLuj93LewUMWl-ZkJ9g/view?usp=sharing

#### Requirements

- Python 3.12
- Kivy 2.3.1

#### Installation

```bash
git clone https://github.com/MdSafwanBinRashid/College_Life_App_KivyPy.git
cd College_Life_App_KivyPy
pip install kivy==2.3.1
python main.py
